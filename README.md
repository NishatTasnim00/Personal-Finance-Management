# PersonalFinanceManagement – Expense Tracker Application

> **A modern, responsive personal finance manager** built with **React, Tailwind CSS, DaisyUI, Firebase Auth, Express.js, and MongoDB**.  
> Track income, expenses, budgets, and detect spending anomalies — all in one place.

![Dashboard Preview]()

---

## Features (Planned)

- **User Authentication** (Firebase) – Secure login & signup  
- **Dashboard** – Real-time balance, cash flow, spending by category  
- **Transaction Management** – Add, edit, delete, recurring transactions  
- **Budget Tracking** – Set limits per category with progress bars  
- **Anomaly Detection** – Flag unusual or large transactions  
- **Reports & Export** – CSV/PDF export of trends and summaries  
- **Responsive UI** – Mobile-first with DaisyUI + Tailwind  

---

## Tech Stack

| Layer         | Technology                          |
|--------------|-------------------------------------|
| **Frontend** | React, Vite, Tailwind CSS, DaisyUI  |
| **State**    | Zustand                             |
| **Forms**    | React Hook Form + Zod               |
| **Auth**     | Firebase Authentication             |
| **Backend**  | Express.js                          |
| **Database** | MongoDB (via MongoDB Atlas)         |
| **Deploy**   | Vercel (Frontend), Render (Backend) |
